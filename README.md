# Exclusive Wallet Premium

Sistema Polygon com RPC Infra Identity - Wallet premium com comunicação criptografada.

## Funcionalidades
- Wallet Polygon Premium
- Sistema Multichain (20 blockchains)
- RPC Infra Identity
- Comunicação criptografada (Chat, Vídeo, Grupos)
- 100% Client-Side
- WebRTC Real

## Estrutura
- `index.html` - Interface principal
- `css/style.css` - Estilos premium
- `js/main.js` - Lógica principal
- `js/identity-dynamic.js` - Identity Dynamic RPC
- `js/blockchains.js` - Gerenciamento de blockchains
- `js/communication.js` - Sistema de comunicação
- `js/webrtc-system.js` - Sistema WebRTC

## Como usar
1. Abra `index.html` em um navegador
2. Conecte com chave privada ou crie nova wallet
3. Explore todas as funcionalidades premium
